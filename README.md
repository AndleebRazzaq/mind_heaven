# Mind Heaven

A **CBT-based AI journaling and cognitive distortion detection** Flutter application with LLM-powered reframing.

Mind Heaven is designed as a **preventive mental wellness** system that analyzes journal text to detect cognitive distortions and emotional tone, then provides structured CBT interventions using a local LLM (Ollama) with confidence-aware safety logic.

## Core Features

- **Context-Aware AI Analysis**
  - Distortion detection (fine-tuned transformer models)
  - Emotion detection (context-mapped states: e.g., "Performance Stress", "Social Anxiety")
- **LLM-Powered CBT Intervention**
  - Personalized insights and pattern explanations
  - Balanced reframes generated by **Llama 3.2**
  - Actionable steps for mental wellness
- **Dynamic Wellness Tools**
  - **Breathing Exercise:** Dynamic card appearing when high stress (в‰Ґ70%) is detected.
  - **Plant Suggestions:** Emotion-based indoor plant recommendations.
- **Insights Dashboard**
  - Weekly mood trends and stress levels derived from journal data.
- **Educational Module**
  - Learn about cognitive distortions and CBT techniques.

## Architecture

The system follows a **3-Layer Intelligence Pipeline**:

1.  **Layer 1: Detection (ML)**
    *   Fine-tuned **DistilBERT** for distortion classification.
    *   **GoEmotions**-based model for raw emotion detection + context mapping.
2.  **Layer 2: Mapping & Context**
    *   Transforms raw labels into human-readable emotional states (e.g., "Decision Anxiety").
    *   Heuristic context detection (Social, Performance, Health, General).
3.  **Layer 3: Intervention (LLM)**
    *   **Ollama (Llama 3.2)** generates structured JSON responses including insights, patterns, and reframes.

## Tech Stack

-   **Frontend:** Flutter (State Management: `provider` + `ChangeNotifier`)
-   **Backend:** FastAPI (Python)
-   **AI Inference:** PyTorch, Hugging Face Transformers, **Ollama**
-   **Database:** Firebase (Auth, Firestore)

## Setup & Running

### 1. Backend Setup

1.  **Install Ollama:** Download and install [Ollama](https://ollama.com/).
2.  **Pull Model:** Run `ollama pull llama3.2:1b` (or your preferred model).
3.  **Install Python Dependencies:**
    ```bash
    cd backend
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    ```
4.  **Run Backend:**
    ```bash
    uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
    ```

### 2. Flutter Setup

1.  **Fetch Packages:** `flutter pub get`
2.  **Configure API:** Ensure `lib/core/config/app_config.dart` points to your backend (e.g., `10.0.2.2:8001` for Android Emulator).
3.  **Enable Remote Mode:** In `lib/app/bootstrap.dart`, ensure `useRemote: true` is set in the `JournalRepositoryImpl`.
4.  **Run App:** `flutter run`
