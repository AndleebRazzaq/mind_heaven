Place indoor plant reference images in this folder for offline display.

Recommended filenames (used by the app):

- `snake_plant.jpg`
- `zz_plant.jpg`
- `lavender.jpg`
- `aloe_vera.jpg`
- `pothos.jpg`
- `peace_lily.jpg`

Mapping by emotion label:

- `anger` -> `snake_plant.jpg`
- `disgust` -> `zz_plant.jpg`
- `fear` -> `lavender.jpg`
- `joy` -> `aloe_vera.jpg`
- `neutral` -> `pothos.jpg`
- `sadness` -> `peace_lily.jpg`
- `surprise` -> `pothos.jpg`

If a local asset is missing, the UI falls back to the backend-provided network image URL.
